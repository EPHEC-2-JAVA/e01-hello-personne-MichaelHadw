Le triangle de Pascal est une présentation des coefficients des termes des binômes de Newton (a + b)n :

  • (a + b)0 = 1
  • (a + b)1 = 1a + 1b
  • (a + b)2 = 1a2 + 2ab + 1b2
  • (a + b)3 = 1a3 + 3a2b + 3ab2 + 1b3

Chaque terme est la somme des deux termes du dessus.

Une ligne n donnée contient n nombres, et le triangle commence par 1 et 1, 1.

Ecrivez une méthode int[][] triangle(int n) qui calcule et retourne ce tableau jusqu’à la ligne n.

Ecrivez ensuite un programme qui demande n à l’utilisateur, puis calculeet affiche le triangle en utilisant deux boucles for ... each imbriquées.
