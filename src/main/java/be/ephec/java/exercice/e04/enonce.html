Le package e04 contient une classe Player.

Comme vous pouvez le voir, un joueur a un nom et un nombre de points accumulés lors d’un jeu.

Le petit programme joint crée deux joueurs, Max et Paul.

Ensuite, ils tirent chacun un dé, celui qui a le plus gros nombre marque un point, jusqu'à ce qu'un d'eux ait atteint le nombre maximum de points..

Utilisez une instruction ternaire pour

  • que la méthode playAndReturnWinner du jeu retourne le gagnant (ligne 21 de Game).
  • ajouter un point au premier ou au deuxième à chaque tour (à la ligne 31 de Game).

Si vous séchez un peu, rappelez-vous que l’opérateur ternaire retourne une valeur…
