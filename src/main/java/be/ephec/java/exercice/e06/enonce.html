Créez une classe Temperature, qui permettra de stocker une température et de l’exprimer aussi bien en degrés Celsius que Fahrenheit. Utilisez une enum pour les unités (TemperatureUnit).

Plus précisément, cette classe devra

  • Posséder un constructeur prenant en entrée la température et l’unité
  • Posséder une méthode getValue(TemperatureUnit) pour renvoyer la température dans l’unité voulue
  • Posséder une méthode setValue(TemperatureUnit) pour modifier la valeur de la température.
  • … et tous les éléments qui vous sembleront nécessaires

Les formules

°C -> °F : f = c * 9 / 5 + 32 (10°C -> 50°F)
°F -> °C : c = (f – 32) * 5 / 9 (23°F -> -5°C)

Note: pour les conversions vous pourriez avoir un problème; si c'est le cas, réfléchissez au type des opérations.
