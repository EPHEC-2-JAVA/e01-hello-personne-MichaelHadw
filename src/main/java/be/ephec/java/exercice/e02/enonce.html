Les méthodes parseInt de conversion de String en Integer acceptent un second paramètre, radix, qui permet de spécifier la base depuis laquelle on interprète un nombre (8, 10, 16 ou même 26).

Les constantes Character.MIN_RADIX et Character.MAX_RADIX permettent de connaître les bases min et max utilisables.

Ecrivez une application Base dans laquelle vous utilisez la classe Scanner pour lire deux entiers; le premier est une base et le second un nombre. Vous affichez ensuite le nombre en base 10. Vous pouvez demander en boucle et arrêter à 0, 0.

  • Exécuter Base avec 10 et 150 donnera 150
  • Exécuter Base avec 16 et 100 donnera 256
  • Exécuter Base avec 16 et CAFE donnera 51966
  • Exécuter Base avec 10 et 150 donnera 150
